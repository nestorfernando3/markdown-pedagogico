# Politica de seguridad

Si detectas una vulnerabilidad, por favor evita publicarla primero en un issue abierto.

## Como reportar

Envia un correo a `nestorfernando3@hotmail.com` con:

- Descripcion tecnica del hallazgo
- Pasos de reproduccion
- Impacto potencial
- Version o commit afectado

## Casos de interes

- Lectura o escritura no autorizada de archivos
- Inyeccion de contenido o escape del sandbox esperado
- Exportacion insegura o exposicion de datos del documento
- Dependencias o configuraciones con riesgo claro para usuarios

Intentare confirmar la recepcion y responder con un plan de manejo responsable.
