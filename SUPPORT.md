# Soporte de Markdown Pedagogico

## Canales recomendados

- **Bugs reproducibles:** GitHub Issues
- **Preguntas sobre instalacion o uso:** GitHub Issues con sistema operativo y version de Node
- **Ideas de producto o feedback pedagogico:** GitHub Issues con contexto de uso

## Incluye esta informacion

- Sistema operativo
- Version de Node
- Si el problema ocurre en modo Vite o Tauri
- Pasos para reproducir
- Mensaje de error, screenshot o stack trace

## Idioma

Se aceptan reportes en espanol o ingles.
